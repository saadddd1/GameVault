# GameVault 编辑精选风重设计 — Design Spec

> **目标：** 将 GameVault 整体 UI 从"AI 味"模板风格改造为编辑精选（Editorial）风，建立统一设计系统，移动优先。

> **技术栈：** Next.js 16 + React 19 + Tailwind CSS v4 + Noto Sans SC + Inter

---

## 品牌

- **Logo:** `GAMEVAULT`（Noto Sans SC Bold 800，字间距 tracking-wide）
- **Tagline:** `游戏 · MOD · 资源`

## 配色

| Token | Hex | Tailwind | 用途 |
|---|---|---|---|
| page-bg | `#FAFAF9` | stone-50 | 页面背景 |
| card-bg | `#FFFFFF` | white | 内容卡片 |
| title | `#1C1917` | warm-black | 主标题、重要文字 |
| body | `#78716C` | stone-500 | 描述文字 |
| brand | `#1E3A5F` | navy-900 | 品牌色、选中态、链接 |
| border | `#E7E5E4` | stone-200 | 卡片边框、分割线 |
| muted | `#A8A29E` | stone-400 | 辅助信息、弱文字 |

## 字体

| 用途 | 字体 | Weight | 备注 |
|---|---|---|---|
| Logo | Noto Sans SC | 800 | letter-spacing: 0.05em |
| 标题 | Noto Sans SC | 700 | |
| 正文 | Noto Sans SC | 400 | line-height: 1.6 |
| 数字/标签 | Inter | 400/600 | 下载数、大小、分类 |

Google Fonts 引入:
```
https://fonts.googleapis.com/css2?family=Inter:wght@400;600&family=Noto+Sans+SC:wght@400;500;700;800&display=swap
```

### GameCard

统一网格卡片，封面 aspect-[4/3]，标题 bold 截断一行，描述截断两行，底部下载数（Inter 数字）+ 大小。

```
┌──────────────┐
│   封面图      │  hover: scale-105
│ [热] [新]    │  标签仅在标记时显示
│         [分类]│  分类角标（可选）
├──────────────┤
│ 标题         │  font-bold, #1C1917
│ 一句话描述    │  text-sm, #78716C, 2行
│ ↓12,345 50GB │  text-xs, #A8A29E
└──────────────┘
```

### Header

- **桌面：** Logo + tagline 左对齐，搜索框居中，导航右对齐
- **移动：** Logo 居中，左侧汉堡菜单图标（展开下拉导航）
- 搜索框 placeholder: `搜索游戏、MOD...`
- 导航链接：首页 / 全部游戏 / MOD工具 / 管理后台（仅管理员可见）

### 分区标题

左侧竖线（品牌色 `#1E3A5F`）+ 标题文字，和正文区分开：
```
▎编辑之选
▎热门下载
▎最近上架
```

### 登录页

- 去掉蓝紫渐变图标，改用纯色品牌色 Logo 字标
- 输入框、按钮匹配新配色（聚焦环用品牌色 `#1E3A5F`）
- 移除测试账号硬编码（已清理）

- `AdminProvider` 认证机制
- 管理后台页面（后续单独处理）
- `next.config.ts`, `tsconfig.json`, `package.json`

---

## 不做什么

- 不添加深色模式
- 不改动管理后台 UI（本次范围外）
- 不加动效/动画（保持克制，编辑风不需要花哨动效）
- 不添加新的 npm 依赖
