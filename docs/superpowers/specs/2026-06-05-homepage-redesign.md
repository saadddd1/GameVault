# 首页重设计

## 目标

精简首页，去掉冗余内容，优化移动端体验。

## 改动范围

### 删除

- 左侧本周/本月/历史最热三栏排行榜
- 右侧"推荐游戏"栏
- 新游推荐轮播区（合并到精选）
- 底部的 footer（page.tsx 内联的那个，保留 Footer 组件）

### 新增/修改

**1. 精选推荐 Hero 区**
- 顶部大图轮播，展示管理员标记为 `isFeatured` 的游戏
- 最多 5 款，少于 1 款时隐藏整个区域
- 桌面端大图横向布局，移动端纵向缩小
- 自动轮播或手动点击切换

**2. 排序栏（融合排行榜）**
- 位置：Hero 下方，游戏网格上方
- 三个选项：下载排行 / 最新上架 / 精选推荐
- 桌面端：下划线 tabs
- 移动端：圆角 pill 标签，可左右滑动
- 默认选中"下载排行"

**3. 游戏网格**
- 桌面端 4 列 → 大屏 5 列
- 移动端 2 列
- 卡片样式保持现有 GameCard

**4. 移动端底部导航**
- 固定底部：首页 / 游戏 / 我的
- 桌面端不显示

### 后端

- Game 模型新增 `isFeatured` 字段（boolean）
- `/api/games` GET 不变，前端自行筛选
- 管理后台添加游戏时可勾选"精选推荐"
- 编辑页面同样支持

### 不复用现有组件的情况

- `app/page.tsx` 大改，但保留 GameCard 组件
- Header/Footer 不变
- AuthProvider 不变
