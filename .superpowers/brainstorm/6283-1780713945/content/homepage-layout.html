首页布局方案

编辑风的核心：不对称、留白、内容分层。哪种布局更符合你的内容优先级？

A

Hero + 网格（接近现状）

顶部大 Hero 精选推荐 → 排序栏 → 游戏网格。结构清晰，用户路径直接。适合"快速找到想玩的游戏"。

B

杂志分区式

精选推荐 → "编辑之选"横向滚动 → "热门下载"列表 → "最近上架"网格。每个板块有独立标题和排版节奏。适合"逛一逛、发现新游戏"。

C

极简信息流

去 Hero，用大号标题 + 一行简介开场 → 游戏以信息流列表呈现（封面左、信息右，像 RSS Reader）。强调内容和数据，最克制。
